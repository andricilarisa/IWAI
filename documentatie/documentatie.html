Rezumat

Interimage este o aplicație web ce interacționează cu utilizatorul prin două tipuri de jocuri: observarea diferențelor din două imagini și un joc de gandire si intuiție ce presupune găsirea de indicii pentru rezolvarea unui mister. Aplicația este una de recreere ce îți stimulează creativitatea și toate simțurile.

Aceasta aplicatie vizeaza orice categorie de utilizator fiind foarte ușor de utilizat prin design-ul său aerisit, user friendly. Jocurile vor avea nivele de dificultate diferită oferind versatilitate.

1. Introducere

Acest raport vine in completarea aplicatiei Interimage si va ajuta utilizatorul sa inteleaga mai bine cum aceasta functioneaza si ce principii foloseste. De ce am creat această aplicație si ce beneficiu aduce societății? Aplicația noastră are ca scop dezvoltarea atenției și a intuiției.
Cum poți combina imaginile in cel mai placut mod dacă nu intr-un joc?
Am dezvoltat 2 tipuri de interacțiuni cu imagini, unul mai dinamic si unul static. In cel static trebuie găsite diferențele dintre 2 imagini, un joc clasic dar care îți dezvolta atenția si îți focusează mintea pe ceea ce ai de făcut. In cel dinamic, trebuie sa găsești ieșirea din capcana pregătită. Cu multă atenție si răbdate din partea utilizatorului, acesta va reuși sa descopere lucruri intereaante si educaționale in același timp.
Dar cum este totul posibil? Cum functioneaza aplicatia? Cum este asigurata siguranta datelor? Pe ce se bazeaza? Acestea sunt doar cateva din intrebarile la care acest raport tehnic va raspunde in continuare.

2. Obiective

Functionalitățile principale ale aplicației noastre este sunt cele doua tipuri de jocuri. Astfel, am inceput dezvoltarea aplicatiei urmarind aceste obiective si principiile de tip AGILE.

Alte obiective urmarite sunt:

  1. Crearea jocurilor cât mai interactive și stimularea intelectului
  2. Crearea unei aplicatii usor de folosit
  3. Invatarea unor noi limbaje de programare
  4. Crearea unei aplicatii responsive astfel incat sa poata fi folosita si de pe telefon
  5. Posibilitatea de a vedea jocul cel mai popular

3. Teorie si metode utilizate

Pe partea de front-end s-au folosit tehnologiile HTML5, CSS3 și JavaScript.

Management-ul aplicatiei s-a facut pe GitHub si GitKraken. GitHub este un serviciu de găzduire web pentru proiecte de dezvoltare a software-ului care utilizează sistemul de control al versiunilor Git. Acesta faciliteaza cmunicarea intre membrii proiectului si are numeroase trasaturi precum bug tracking, task management ce ajuta dezvoltatorii in implementare.

GitKraken este un Git GUI ce faciliteaza utilizarea GitHub-ului.

Limbajul ce il vom folosi în crearea jocului este JavaScript.

4. Cum functioneaza?

Find The Difference
Jocul presupune găsirea diferențelor dintre două imagini. Se definesc anumite zone de interes (elemente 2D) suprapuse peste imagine pe care utilizatorul va putea da "click" declanșând o acțiune ce va modifica proprietațile vizuale. Astel diferențele vor dispărea de pe imagine.
Escape Room
Jocul prsupune găsirea de indicii și rezolvarea unui mister. Utilizatorul are posibilitatea de a da "click" pe mai multe elemente indiferent daca sunt indicii sau nu. Acțiunea va determina selectarea anumitor obiecte și posibilitatea de a le utiliza mai târziu.
